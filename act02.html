Exercice de rappel

  1. Générer un nombre aléatoire entre 5 et 25.
  2. Simuler le lancement de deux dés à six faces.
  3. Améliorer votre programme pour qu'il utilise des images au lieu des nombres.

    Résultat attendus

Télécharger les images dans votre dossier de travail.

Question n°1 - réponse

<script>
  // Génère un nombre aléatoire dans l'intervalle [a, b]
  function randint(a, b) {
    return Math.floor(Math.random() * (b - a + 1)) + a;
  }
  // Générer un nombre puis l'afficher
  let v = randint(5, 25);
  document.writeln("<p>Nombre aléatoire [5, 25] : ", v, "</p>");
</script>

Application

Ecrire un script qui saisit puis affiche la somme de deux nombres en utilisant la boite d'invite.

<div id="msg"></div>
<script>
  // Méthode 1 : boîte d'invite
  let a1 = Number(prompt("Donner a ? ", "6"));
  let b1 = Number(prompt("Donner b ? ", "6"));
  let s1 = a1 + b1;
  const msg = document.getElementById("msg");
  msg.innerHTML = a1 + " + " + b1 + " = " + s1;
</script>

Application

Ecrire un script qui lit la valeur de deux champs numériques a et b, puis affiche leurs sommes.

Le formulaire est comme suit :

<form action="#" method="post" name="f">
  <div><label for="a">A</label>
    <input type="number" name="a" id="a" value="9"></div>
  <div><label for="b">B</label>
    <input type="number" name="b" id="b" value="16"></div>
  <div><label for="r">Résultat</label>
    <input type="number" name="r" id="r" disabled></div>
</form>
<script>
  let a = Number(document.f.a.value);
  let b = Number(document.f.b.value);
  let s = a + b;
  document.f.r.value = s;
</script>

Application

Ecrire un script qui demande à l'utilisateur s'il est stressé ou non ?

  • Si l'utilisateur est stressé : il lui demande de manger du chocolat,
  • Si l'utilisateur n'est pas stressé : il lui demande de contiuner de sourire.
<script>
const stressed = confirm("Are you stressed ? ");
if (stressed) {
  alert("Eat chocolate");
} else {
  alert("Keep smiling");
}
</script>
